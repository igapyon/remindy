# PikoMouseCommand.java

## Overview
スクリーンセーバーやスリープ突入を抑止するため、マウスポインタを毎分わずかに動かすユーティリティコマンド。`Robot` を使い 1 ピクセル分だけ上下左右のいずれかへ移動させます。

## Behavior
- `MouseInfo.getPointerInfo()` で現在位置を取得し、`Random` で決めた方向に 1px 移動。
- 例外（Robot 未対応やアクセス拒否）は捕捉し、標準エラーに簡潔なメッセージを残すのみ。

## Notes
- 入力フォーカスに影響しないよう移動量は最小限 (1px)。
- `MinuteTicker` への登録順次第で、通知と同じタイミングに合わせたサイクルを維持できます。
