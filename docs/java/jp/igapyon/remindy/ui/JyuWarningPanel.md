# JyuWarningPanel.java

## Overview
警告表示用にカスタマイズした `JPanel`。黒地に赤ストライプと太線を描画し、中央に大きめフォントで複数行テキストをセンタリングします。

## Rendering Flow
- `paintComponent` をオーバーライドし、背景・ストライプ帯・上下ラインを順に描画。
- メッセージを `\n` で分割し、使用可能な中央領域に対してバーティカルセンタリング。
- アンチエイリアスを有効化し、`MS Gothic` 太字 40pt で警告テキストを描画。

## Notes
- `drawStripeBand` で傾いた帯を繰り返し描くことで警戒感を演出。
- `JyuWarningPopup` の内容ペインとして使用されますが、独自のダイアログに組み込むことも可能です。
