# 依存ライブラリ

このプロジェクトは、以下のサードパーティ製ライブラリを利用しています。

これらのライブラリはすべて Apache License, Version 2.0 の下でライセンスされており、本プロジェクトのライセンスと互換性があります。

詳細なライセンス情報については、ビルド時に生成される `target/generated-resources/licenses/THIRD-PARTY.txt` ファイルを参照してください。

## 主な依存ライブラリ

- **Jackson**: JSONデータの処理に使用されます。
- **Apache Commons CSV**: CSVデータの処理に使用されます。
